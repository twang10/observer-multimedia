# Long-Distance Relationship Infographic
LDR survey results page made for the Tufts Observer's Love issue. The page can now be viewed live at http://tuftsobserver.org/multimedia/longdistance/

This page was made using skrollr.js to create animated scrolling affects using CSS animations. Icons are from Olha Filipenko on Behance and the animated graph was adapted from http://codepen.io/enxaneta/pen/KVGWQL
